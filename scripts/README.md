# Scripts

## Inspection

- PHPMD
- PHPCS
- PHPDPD
- SensiLabs Security Checker

Inspections will be stored in ./build directory

```sh
sh ./scripts/inspect.sh
```

## Create data directory with childs

sh ./scripts/mkdata.sh
```

sh ./scripts/generate_maps.sh
```
